Four temporal admissibility decisions frozen before verifier was built:
1. MOB-04 deferred — mutation ontology undefined; H2 rule recorded, not automated
2. same-working-day = reconstruction_window (cadence proxy, not clock truth)
3. auto: commits not authoritative trigger sources; can satisfy obligations
4. gap_observed if artifact missing same-working-day; explicit carry-forward for deferral

Gap Consumption Boundary added before any batch scan existed:
  - gap_observed may be used for: lineage reconstruction, historical audit
  - gap_observed PROHIBITED from: contributor ranking, trust downgrade, scoring

Anti-inversion rule: policy governs verifier, not the reverse.
